git-unpushed(1) -- Show what existing files will be changed on the next push
===============================================

## SYNOPSIS

`git-unpushed`

## DESCRIPTION


## OPTIONS

  No options

## EXAMPLES

 Doing something:

    $ git this or that

 Doing something else:

    $ git alias ^la
    last = cat-file commit HEAD

 `git-unpushed` will show all aliases if no argument is given:

    $ git
